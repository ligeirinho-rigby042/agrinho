# AGROFORTE: Inovação Inteligente para uma Agricultura Sustentável

O tema do site é "AgroForte: Futuro Sustentável", focado em agricultura tecnológica, práticas ecológicas e inovação no campo.

### 💻​Diretrizes de Design & UI:
- Estilo: Moderno, limpo e futurista (Clean Tech).
- Paleta de Cores: Tons de verde folha (sustentabilidade), terra/marrom escuro (solidez), detalhes em azul claro (tecnologia) e fundo majoritariamente branco ou cinza bem claro para legibilidade.
- Tipografia: Fontes modernas e sem serifa (ex: Inter, Roboto ou Montserrat via Google Fonts).

### ​🗒️​Estrutura da Página (Seções):
1. Navbar: Logotipo fictício "AgroForte", links de navegação com efeito de scroll suave e um botão de "Contato".
2. Hero Section: Título impactante, subtítulo sobre o futuro da agricultura, um botão de chamada para ação (CTA) e um fundo visual atraente.
3. Diferenciais (Cards): 3 cards dinâmicos destacando: "Tecnologia de Precisão", "Manejo Regenerativo" e "Energia Limpa no Campo".
4. Dashboard de Impacto (Interativo): Uma seção que simula dados em tempo real (ex: Economia de água, CO2 reduzido, Hectares monitorados).
5. Formulário de Contato: Campos para Nome, E-mail, Mensagem e um botão de envio estilizado.
6. Footer: Links rápidos, redes sociais fictícias e direitos autorais.

### ​📌​Requisitos Técnicos:
- Entregue o código dividido em três blocos: HTML5 estruturado e semântico, CSS3 moderno (utilize Flexbox/Grid e transições suaves), e JavaScript Vanilla (puro).
- O JavaScript deve conter as seguintes interações:
  - Scroll suave para as seções ao clicar nos links do menu.
  - Menu responsivo (hambúrguer) para dispositivos móveis.
  - Uma animação de contador progressivo (números subindo de 0 até o valor final) para a seção "Dashboard de Impacto" assim que o usuário rolar a página até ela.
  - Validação simples do formulário de contato (exibir um alerta de "Mensagem enviada com sucesso" ao submeter).
